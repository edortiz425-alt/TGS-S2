# Ficha tecnica del proyecto

## Datos generales

- **Proyecto:** Objeto Virtual de Aprendizaje (OVA)
- **Tema:** 6. Propiedades de los sistemas
- **Autor:** Gelves Triana Sebastian
- **Asignatura:** Teoria General de Sistemas
- **Formato:** pagina web interactiva para localhost y publicacion web

## Objetivo general

Disenar e implementar una OVA funcional, accesible y didactica que explique las propiedades de los sistemas y permita practicar el tema mediante recursos interactivos.

## Objetivos especificos

- Sintetizar el marco conceptual de las propiedades de los sistemas.
- Relacionar conceptos como sinergia, totalidad, retroalimentacion, emergencia, entropia, limites, entorno, jerarquia y equifinalidad con casos reales.
- Incorporar ejercicios con retroalimentacion, juegos y quiz automatico.
- Preparar el proyecto para ejecucion local y publicacion en una plataforma web.

## Herramientas usadas

- Python 3 para servidor local.
- HTML5, CSS3 y JavaScript.
- Canvas para simulaciones visuales.
- SVG local como mapa conceptual.
- Apoyo de IA para generacion y organizacion del codigo.

## Componentes de la OVA

- Contextualizacion y justificacion.
- Marco teorico con referencias academicas.
- Ejemplos aplicados a hospital, transporte urbano y tienda en linea.
- Simulador interactivo de conectividad, retroalimentacion y ruido.
- Glosario buscable para repasar conceptos durante la sustentacion.
- Mision integradora "Campus Vivo" con diagnostico, modelo iceberg y radar sistemico.
- Ejercicios con retroalimentacion inmediata.
- Tres juegos interactivos.
- Tres videos reales de YouTube con enlaces directos verificados.
- Quiz con puntaje automatico.
- Pasaporte de progreso y credencial imprimible.
- Creditos y referencias.
