OVA: Teoría de Sistemas (Teoría de Colas y Juegos)
Objeto Virtual de Aprendizaje (OVA) diseñado para la enseñanza práctica de la Investigación de Operaciones. Este entorno interactivo permite explorar los fundamentos de la Teoría de Colas y la Teoría de Juegos mediante simulaciones en tiempo real.

Descripción del Proyecto
El OVA ayuda a entender cómo gestionar sistemas complejos y la toma de decisiones estratégicas. A través de simuladores, los estudiantes pueden visualizar y manipular variables matemáticas para comprender fenómenos reales como:

Teoría de Colas: Gestión de líneas de espera, optimización de recursos y disciplinas de servicio (FIFO, LIFO, Prioridad).

Teoría de Juegos: Toma de decisiones bajo interdependencia, Equilibrio de Nash, estrategias competitivas y dilemas sociales (como el Dilema del Prisionero).

Características Principales
Simuladores Interactivos: Entornos prácticos para aplicar la teoría (ej. gestión de sucursales bancarias, optimización de puertos y dilemas viales).

Métricas en Tiempo Real: Visualización del comportamiento del sistema según las variables ajustadas por el usuario.

Evaluación: Incluye laboratorios interactivos y cuestionarios para medir el aprendizaje.

Base Teórica: Referencias académicas sobre sistemas abiertos, modelos de espera y equilibrio estratégico.
