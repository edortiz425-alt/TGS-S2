# OVA_Dexly
es una herramienta interactiva diseñada para explorar la teoría de las Organizaciones Inteligentes de Peter Senge. A través de un enfoque basado en el Pensamiento Sistémico, el usuario podrá comprender cómo la integración de las cinco disciplinas —Dominio Personal, Modelos Mentales, Visión Compartida, Aprendizaje en Equipo y Pensamiento Sistémico—
